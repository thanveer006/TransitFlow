TransitFlow AIThe Smart Mobility Orchestrator for Urban KeralaTransitFlow AI is a premium, real-time decision-support system designed to rescue private vehicle commuters (cars/bikes) from unpredictable urban gridlock and weather challenges in Kerala's major hubs like Kochi, Trivandrum, and Kozhikode.1. Problem StatementUrban commuters in Kerala face two major unpredictable enemies: Sudden Tropical Rain and Extreme Traffic Bottlenecks.The Gap: Standard navigation apps show traffic but don't tell you when to abandon your vehicle in favor of public transit to save time.The Friction: For bike riders, sudden rain isn't just a delay—it’s a safety hazard and a threat to professional grooming (especially when heading to interviews or meetings).The Conundrum: Drivers often hesitate to switch to Metro or Rail because they lack real-time data on parking availability and total cost comparison.2. Project DescriptionTransitFlow AI acts as a "Dynamic Escape Hatch." It monitors your private commute and identifies the exact "Switch Point" where public infrastructure becomes superior to staying in your car.Core Functionality:Predictive Rerouting: If the AI detects that $Driving\_Time > (Transit\_Time + 10 mins)$, it triggers a high-priority intervention.Environmental Intelligence: Real-time "Rain Circles" on the map identify waterlogged areas. The AI then suggests AC Metro or Train routes to keep the user dry and safe.Multi-Modal Integration: Seamlessly stitches together Google Maps Directions for private travel with Metro and Rail schedules.Fare-Link: A live financial dashboard comparing fuel/parking costs vs. public transit fares to encourage sustainable choices.3. Google AI IntegrationTools & Models UsedGemini 3 Flash: The "Brain" of the application, used for route reasoning, weather impact analysis, and generating context-aware voice scripts.Google Maps Platform: * Directions API: For real-time polyline rendering and route calculations.Places API (Autocomplete): For smart location suggestions across Kerala.Traffic Layer: For real-time bottleneck identification.Web Speech API: To deliver Gemini’s AI-generated navigation logic via hands-free voice commands.How AI is IntegratedDynamic Reasoning: Unlike static maps, Gemini analyzes why you should switch. It generates scripts like: "Heavy waterlogging at Edapally is causing a 20-minute delay. Switch to Metro Line 1 now to reach your 10:00 AM meeting on time."Contextual Awareness: When 'Interview Mode' is toggled, the AI prioritizes "Freshness" over speed, suggesting AC transit modes even if the time saving is marginal.Synthetic Data Simulation: Gemini simulates "Parking Spot Availability" at Metro stations based on current time-of-day traffic flow to reduce "Last-Mile" anxiety.4. Features at a GlancePremium Dashboard: High-contrast #050505 dark theme with #8B5CF6 glassmorphism UI.Live Rain Radar: Circular visual overlays on the map showing localized rain zones with particle effects.Simulate Bottleneck: A one-click demo feature to trigger a traffic surge and watch the AI reroute in real-time.Search Autocomplete: Specialized search restricted to Kerala for faster, more accurate destination picking.5. Clone the repository
git clone https://github.com/your-username/transitflow-ai

# 2. Navigate to the folder
cd transitflow-ai

# 3. Install dependencies
npm install

# 4. Configure Environment Variables

VITE_GOOGLE_MAPS_API_KEY=your_google_maps_key

# 5. Launch the development server
npm run dev
