🚀 TransitFlow AI
The Smart Mobility Orchestrator for Urban Kerala

TransitFlow AI is a premium, real-time decision-support system designed to help private vehicle commuters (cars/bikes) escape unpredictable traffic and weather challenges in Kerala’s major cities like Kochi, Trivandrum, and Kozhikode.

🧠 Problem Statement

Urban commuters in Kerala face two major unpredictable challenges:

🌧️ Sudden Tropical Rain
🚗 Extreme Traffic Bottlenecks
❗ The Gap

Standard navigation apps show traffic but don’t tell you when to switch to public transport.

⚠️ The Friction
Bike riders → rain = safety + appearance issues
Drivers → unsure about parking + cost comparison
🤯 The Conundrum

People hesitate to switch to Metro/Rail due to:

No real-time parking data
No cost comparison
💡 Project Description

TransitFlow AI acts as a Dynamic Escape Hatch 🚀

It monitors your journey and identifies the perfect “Switch Point” where public transport becomes better than private travel.

⚙️ Core Functionality
🔁 Predictive Rerouting
If Driving_Time > (Transit_Time + 10 mins)
→ AI triggers switch recommendation
🌧️ Environmental Intelligence
Detects rain zones (“Rain Circles”)
Suggests AC Metro/Train routes
🔗 Multi-Modal Integration
Combines:
Private travel (car/bike)
Public transport (Metro, Rail)

🚀 Setup Steps
# 1. Clone the repository
git clone https://github.com/your-username/transitflow-ai

# 2. Navigate to project
cd transitflow-ai

# 3. Install dependencies
npm install
🔐 Environment Variables

Create .env file:

VITE_GOOGLE_MAPS_API_KEY=your_google_maps_key
VITE_GEMINI_API_KEY=your_gemini_api_key
▶️ Run Project
